E_Garage – Online Vehicle Service Management System
📌 Project Overview:
    - E_Garage is a web-based application built using Python & Django that allows users to book vehicle services online.
    - It simplifies garage management by enabling service booking, tracking, and admin control in one platform.

Tech Stack:
    - Frontend: HTML, CSS, JavaScript
    - Backend: Python (Django Framework)
    - Database: PostgreSQL
    - Tools: Git, GitHub, VS Code

Step 1: Clone Repository
    - git clone https://github.com/your-username/E_Garage.git
    - cd E_Garage
    
E-Garage Project Steps:
-----------------------
1.open vscode
2.open terminal in cmd ,pip install django
3. python -m venv venv
4. venv\Scripts\activate
5. python manage.py makemigrations
6. pip install psycopg2
7. pip install psycopg2-binary
8. python manage.py makemigrations
9. python manage.py migrate
10. python manage.py runserver
